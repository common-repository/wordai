== Installation ==
1. Install 'WordAI' Auto Content Writer by uploading the plugin to the '/wp-content/plugins/' directory inside.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. First put your openAI [API key](https://platform.openai.com/account/api-keys) through 'API Settings' page of the plugin.
4. Then you are ready to generate auto content / HD images for your Wordpress!

== Frequently Asked Questions ==
= What is "WordAI" plugin? =
"WordAI" is Auto Content Writer plugin uses [OpenAI](https://openai.com/) Artificial Intelligent technology by processing natural language to generate high-quality content/images as like human. It will help you to write fast SEO content/images automatically based on your input word/words within seconds.

= Generated Titles / Content are look like natural as like human creation?=
Yes, all created Titles/Content are very human friendly, relevant and natural as like human creation. 

= How does the "WordAI" plugin generating content/images? =
The plugin uses advanced latest Artificial Intelligent(AI) and natural language processing techniques based on OpenAI to generate high-quality, SEO friendly unique content/images. 


= Is generated content/images are Unique? =
Yes, all generated content/images are natural, human friendly & unique. Titles/Content will generate randomly based on your input word/words.

= Can I use the plugin to generate content other languages than English? =
Yes, plugin supports to generate content the language you want by selecting the language option in 'Content Settings' menu page. Plugin supports all languages.

= Can I use the latest model to generate content / images? =
Yes, WordAI supports the latest GPT-4o model. Also supporting GPT-4o mini, most cost-efficient small model.

= Can I limit the content paragraphs? =
Yes, WordAI supports how many paragraphs you want to write, in 'Content Settings' menu page select the drop down menu 'Content Paragraph number' option. 

= Can I limit the generated image number with image size? =
Yes, WordAI supports to set image sizes with image number limit, in 'Image Settings' menu page you can set those options. 

= Why my result text / content are cut off / truncated ? =
If this happens then you have changed / set API parameters wrongly - just go to WordAI menu -> API Settings, click on 'Reset Settings' button to set all API parameters as default & check to see your results now. You may change ChatGPT model or other parameters as needed.

= Why I do not see any output generated text / content ? =
You may have issue with streaming feature with your hosted server restriction / settings, You just uncheck the 'Streaming' option through WordAI menu -> API Settings, and then click 'Save API Settings' button to save the changes and check the result.
